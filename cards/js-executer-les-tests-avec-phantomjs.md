# Exécuter les tests avec PhantomJS

## Headless WebKit with JavaScript API

```
phantomjs  TEST_RUNNER.js  TEST_OUTPUT.html
```

A combiner avec Gradle et Jasmine (ou Mocha, QUnit) pour automatiser l'exécution des tests.
